Welcome to Cards Against Humanity!

Creators, Victor Veytsman and Justin Kim




Purposes in making CAH

This game is created in the purposes of creating a simple game to program and interact with during the 2019 StuyHacks


How to use

Installation

You must have Go and Python3 installed 

$ #Install required python dependencies
$ python3 -m pip install colorama websocket_client
$ #Install required go dependency
$ go get github.com/gorilla/websocket
$ #Get and install CAH
$ git clone https://github.com/vityavv/asdf
$ cd asdf/network
$ go install

Usage

Once you install it, run the command network in the asdf/network folder to run the server.
To run the client, run the file CAH.py in the asdf/interface folder with the command python3 CAH.py. Then, follow what it says on the screen.
